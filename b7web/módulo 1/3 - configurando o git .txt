Configurando informações de usuário:

git config --global user.name "Seu Nome": Define o nome do usuário globalmente.
git config --global user.email "seuemail@example.com": Define o email do usuário globalmente.
Configurando ferramentas de merge e diferenças:

git config --global merge.tool <tool>: Define a ferramenta de merge a ser usada (ex: vimdiff, kdiff3, etc.).
git config --global diff.tool <tool>: Define a ferramenta de visualização de diferenças a ser usada.
Configurações do editor:

git config --global core.editor <editor>: Define o editor padrão para mensagens de commit (ex: vim, nano, subl, etc.).
Configuração de cores:

git config --global color.ui true: Ativa a colorização da saída do Git.
Configuração de atalhos (aliases):

git config --global alias.<alias-name> <git-command>: Cria atalhos para comandos frequentemente usados (ex: git config --global alias.co checkout para git co).
Configuração de repositórios remotos:

git remote add <nome-remoto> <url-remoto>: Adiciona um repositório remoto ao seu projeto.
git remote -v: Exibe os URLs dos repositórios remotos configurados.
Configuração de fluxo de trabalho:

git config --global pull.rebase true: Configura o Git para usar rebase ao puxar (em vez de merge).
git config --global push.default <simple|current|upstream|matching>: Define a política de push padrão.
Configuração de ignorar arquivos (.gitignore):

git config --global core.excludesfile <caminho-para-gitignore>: Configura um arquivo global .gitignore.
Configuração de comportamento de final de linha:

git config --global core.autocrlf <true|false|input>: Lida com a conversão automática de terminações de linha.
Lembre-se de que muitas dessas configurações também podem ser ajustadas no nível do projeto, não apenas globalmente. Use git help config para mais informações e detalhes sobre as configurações disponíveis.
